# university-academic-tracker – Scope

Este documento forma parte de la documentación del proyecto **University Academic Tracker**.  
Su propósito es dejar constancia del **alcance general** del sistema durante el semestre, de manera resumida.

---

## Alcance General

El proyecto se centrará en:

- **Gestión de calificaciones**: materias, notas por actividad y cálculo de promedios.  
- **Configuración dinámica de porcentajes de corte.**  
- **Organización académica básica**: registro de actividades y entregas por corte.  
- **Autenticación de usuarios** con roles básicos (estudiantes y docentes).  

De forma **opcional**, y solo si el tiempo lo permite (12 semanas), se incluirán:  
- Reportes básicos en PDF.  
- Alertas simples para recordatorios.
