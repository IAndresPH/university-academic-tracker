ÉPICA 1: Control del rendimiento académico



Sprint 1

HU1: Como estudiante, quiero registrar el nombre, código y número de créditos de cada materia, para poder llevar un control académico organizado.

HU2: Como estudiante, quiero registrar las actividades (parciales, talleres, proyectos) asociadas a cada materia, indicando su porcentaje de evaluación, para organizar el plan de notas del semestre.



Sprint 2

HU3: Como estudiante, quiero ingresar las calificaciones obtenidas en cada actividad, para llevar un registro exacto de mis notas.

HU4: Como estudiante, quiero que el sistema calcule automáticamente el promedio de cada corte, usando las notas ingresadas y los porcentajes definidos, para conocer mi rendimiento parcial.



Sprint 3

HU5: Como estudiante, quiero visualizar en un panel el promedio final de cada materia al finalizar todos los cortes, para conocer mi resultado definitivo.

HU6: Como estudiante, quiero que el sistema me muestre la nota mínima necesaria en el próximo corte, para alcanzar una meta de promedio definida por mí.
